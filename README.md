# 💬 WhatsApp Outreach Message Generator

An AI-powered tool to generate personalized cold outreach messages for WhatsApp leads.

## ✨ Features

- 📤 Upload CSV with business leads
- 🤖 AI-generated personalized messages using OpenAI
- 📊 Interactive web interface with Streamlit
- 📥 Download results as CSV
- 🎨 Multiple message styles
- 🔧 Adjustable creativity (temperature)
